Database Settings

Firebase Realtime sync

By default, the application runs on a shared secure Firebase/Firestore instance dynamically created inside this workspace. This works instantly on mobile and desktop without any manual registration!

Perfect for your Wedding! Connecting to Supabase allows you to manage the wedding message board entirely in your own clean control panel. 
CREATE TABLE messages (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  guest_name text NOT NULL,
  relationship text NOT NULL,
  message_body text NOT NULL,
  loves integer DEFAULT 0,
  created_at timestamp with time zone DEFAULT timezone('utc'::text, now()) NOT NULL
);

-- Enable Realtime for live updates (Optional but highly recommended)
alter publication supabase_realtime add table messages;
